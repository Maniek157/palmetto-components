# Duration

Use 'Duration' to display a time span in a friendly human readable format. The `Duration` component handles converting 
between different time units at defined intervals, including rounding, and displaying the result with an approperiate label.

<Canvas>
  <Duration milliseconds={1000} /> 
  <Duration milliseconds={60000} /> 
  <Duration milliseconds={7200000} /> 
  <Duration milliseconds={86400000} />
</Canvas>

## Steps

The steps at which the `Duration` component switches between time units can be configured using the 
`displaySecondsUntilInMs`, `displayMinutesUntilInMs`, and `displayHoursUntilInMs` properties.

<Canvas>
  <Story name="Steps">
       <Duration milliseconds={3600000} displayMinutesUntilInMs={3600000} /> 
  </Story>
</Canvas>  

## Rounding

By default the `Duration` component will round up the time span to the nearest whole unit. This can be configured using the
`roundUp` property.

<Canvas>
  Up: <Duration milliseconds={60001} roundUp={true} /> 
  Down: <Duration milliseconds={60001} roundUp={false} as="p" /> 
</Canvas>
